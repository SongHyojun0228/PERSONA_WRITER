# 🚀 Project: Persona Writer (페르소나 라이터)

> AI 기반 웹소설 창작 어시스턴트 및 설정 관리 SaaS설정 오류 없는 완벽한 집필
AI 디지털 파트너와 함께하는 서사 혁명
**"당신 안의 이야기가 세상에 나올 시간. 페르소나 라이터와 함께라면 누구나 작가입니다.”**
> 

---

## 1. 프로젝트 비전

- **핵심 가치**: 작가가 설정 붕괴 걱정 없이 집필에만 집중할 수 있는 환경 제공
- **목표**: AI를 통해 캐릭터의 생동감과 서사의 긴장감을 극대화하는 '디지털 집필 파트너' 구축

---

## 2. 상세 기능 명세 (Detailed Features)

### 🟢 A. AI 캐릭터 엔진 (Character Engine)

- **초정밀 시트 생성**: 이름, 나이, 성별, 외모, 성격(MBTI), 핵심 트라우마, 숨겨진 비밀 관리
- **페르소나 대사 시뮬레이터**: 특정 상황(배신, 대치 등) 입력 시 캐릭터 설정에 기반한 고유 말투 대사 3종 추천
- **인물 관계도 시각화**: 인물 간 관계(애정, 증오 등)를 그래프 뷰로 시각화

### 🔵 B. 스마트 에디터 & 복선 트래커 (Smart Editor)

- **실시간 글자 수 카운터**: 웹소설 표준(5,500자) 기준 게이지 바 표시
- **인라인 복선(떡밥) 마킹**: 텍스트 드래그 후 `#복선` 등록 시 사이드바 자동 저장 및 회수 체크 기능
- **설정 위키 팝업**: 집필 중 단어/이름 클릭 시 즉시 설정 정보 팝업 표시

### 🟡 C. 서사 분석 및 피드백 (Story Analysis)

- **전개 속도(Pacing) 체크**: AI의 회차 분석을 통한 속도감 피드백 (급함/지루함 등)
- **캐릭터 일관성 검사**: 설정된 성격과 대사의 불일치 실시간 경고
- **맞춤법 및 문장 교정**: 웹소설 특화 구어체/문어체 구분 교정 제안

### 🔴 D. 비주얼 & 부가 기능

- **AI 표지 프롬프트 메이커**: 로그라인 + 외모 설정을 조합한 최적화된 영어 프롬프트 생성
- **다크 모드 지원**: 장시간 집필 작가를 위한 필수 UI
- **클라우드 동기화**: Supabase 활용 실시간 자동 저장

---

## 3. 기술 스택 및 데이터 구조

### 💻 Tech Stack

- **Frontend**: React.js (Vite), Tailwind CSS, Lucide-react
- **Editor**: Tiptap Editor (Headless 커스터마이징)
- **Backend**: Supabase (Auth, Database, Storage)
- **LLM**: Gemini 1.5 Pro/Flash API (결제 등급에 따른 최적화)

**📊 Database Schema (핵심 테이블)**

| **테이블** | **필드 구성** |
| --- | --- |
| **novels** | `id`, `user_id`, `title`, `genre`, `logline`, `cover_url` |
| **chapters** | `id`, `novel_id`, `chapter_num`, `content`, `word_count`, `memo` |
| **characters** | `id`, `novel_id`, `name`, `personality`, `trauma`, `speech_style`, `metadata(JSON)` |
| **foreshadows** | `id`, `novel_id`, `content`, `status(open/closed)`, `created_at` |

## 5. 수익화 및 확장 전략

### 5.1 추가 고도화 기능

- **플롯 생성 어시스턴트**: 로그라인 기반 기승전결 시놉시스 제안
- **버전 관리(Git for Writers)**: 수정 전 버전으로 쉽게 복구하는 시스템
- **모바일/태블릿 지원**: 반응형 웹 강화로 어디서든 집필 가능

### 5.2 멤버십 모델 (Monetization)

- **Free Tier**: 기본 에디터, 프로젝트 제한, AI 채팅 일일 횟수 제한
- **Premium Tier (구독)**: 고급 AI 기능(플롯/퇴고) 무제한, **Gemini 1.5 Pro** 모델 선택권, 마인드맵 자동 생성
- **콘텐츠 플랫폼**: 완성작 유료 발행, 독자 후원 시스템 및 에세이 구독 전송 서비스(메일/카톡)
